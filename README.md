# Site Fit House Academia

Site institucional responsivo da **Fit House Academia**, localizada em Anita Garibaldi, Santa Catarina.

## Executar no VS Code

1. Extraia completamente o arquivo ZIP.
2. Abra a pasta `fit-house-site` no VS Code.
3. Abra o terminal e execute:

```bash
npm install --no-audit --no-fund
npm run dev
```

Acesse no navegador:

```text
http://localhost:5173
```

O projeto não utiliza dependências externas de JavaScript. Por isso, o `npm install` deve terminar quase instantaneamente.

Também é possível abrir o arquivo `INICIAR.bat` no Windows.

## Criar a versão de publicação

```bash
npm run build
```

Os arquivos prontos serão copiados para a pasta `dist`.
